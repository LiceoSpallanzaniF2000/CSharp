#CSharp

##Ciclo if

###Sintassi

```visual studio
	if (*condizione*)  
	{  
		...  
	}  
	else if (*condizione*)  
	{  
		...  
	}  
	else  
	{  
		...  
	}  
```

###Ricorda
* i simboli per i confonti sono  
	* a > b  
	* a < b  
	* a >= b  
	* a <= b  
	* a == b  
	* a != b  
* and --> &&  
* or  --> ||
